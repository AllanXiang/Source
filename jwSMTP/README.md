## 简介
jwSMTP是一个由C++编写的邮件发送库，支持Linux、Windows等平台。可使用HTML或纯文本方式发送邮件,                                                     也可添加附件，支持多个收件人，并且支持LOGIN和PLAIN两种服务器验证方式，此代码可以帮助简单理解SMTP协议和DNS协议。
